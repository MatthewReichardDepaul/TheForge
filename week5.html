Go Back

This week, we were tasked with using the 3D printers. We had to first do a test print (of some design that somebody else made) so that we know how to use the equipment. Then, we had to create our own 3D design that could not be made subtractively (i.e. with a CNC mill) and print it. We also used a 3D scanner to scan something.



Articulated Sausage

For my test print, I found this file for 3D articulated sausages made by DeadCat1337 on Thingiverse. I loaded the file into a slicer software and printed it on a Creality CR-10S, and it turned out really well. It was able to move freely after removing the support structures from the print.



Propeller Drawer

For the model I designed myself, I wanted to make a small desktop drawer I could put random things in, and I decided to add a spinning propeller to the top for no real reason. The reason why this design is not possible subtractively is because it was all done 'print-in-place' meaning that the drawer was printed inside the housing, the propeller was printed on the axel, everything was printed all at once with no assembly required. This is not possible with a subtractive process. Designing the model was fairly simple, so I loaded it into the slicer and printed this time using the Bambu Labs X1-Carbon printer which was extremely fast. The first attempt didn't go so well, which you can see pictured, but after that it turned out perfectly. Removing the support structures was a real pain (you can see a pile of stringy support structures which I removed from it), but everything worked as expected and turned out great.
